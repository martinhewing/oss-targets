# OSS Contribution Targets

Open source projects where our daily usage of the ConnectionSphere voice AI stack gives us an unfair advantage as contributors.

Every entry maps to code we actually wrote, a workaround we built, or a gap we hit while building the Factory's Cartesia + Claude + FastAPI voice pipeline.

**Live:** [martinhewing.github.io/oss-targets](https://martinhewing.github.io/oss-targets)

## Development

```bash
npm install
npm run dev
```

## Deploy

Push to `main`. GitHub Actions builds and deploys to Pages automatically.
